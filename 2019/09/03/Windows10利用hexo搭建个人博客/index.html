Windows 10 利用hexo搭建个人博客

一、搭建所需

1、系统：Windows 10

2、软件需求：hexo、node.js、GitHub

二、软件安装

1、安装node.js、github、hexo（按此顺序安装）

2、先安装node.js和git，后安装hexo（https://hexo.io/zh-cn/docs/）：打开命令提示符（管理员模式）窗口，输入命令：
npm install -g hexo-cli

3、查看各软件版本号：node -v/ hexo -v /git -version

三、Git部署

1、配置git用户信息（桌面空白处，右击鼠标，选择Git Bash Here）
git config –global user.name “GitHub用户名”
git config –global user.email “GitHub邮箱”
例子：
git config –global user.name “xx-yyzz”
git config –global user.email “1234567890@qq.com“

2、生成ssh密钥文件，注意，需要按三次回车键。第一次回车键，选择默认文件夹存放生产的秘钥文件；第二次回车，代表密码为空（不设密码）；第三次回车，确认不设置密码。
ssh-keygen -t rsa -C “Github邮箱”

3、前往github创建个人仓库，仓库名字必须为：username.github.io ,将刚新生成的公钥id_rsa.pub(用记事本打开)添加到Github你创建的仓库中，推荐使用QQ浏览器（网页翻译功能）。完了以后使用命令ssh git@github.com测试

四、博客搭建

1、选择一个盘创建一个文件夹“Myblog”
2、在Windows PowerShell（管理员）模式下进入Myblog文件夹
3、初始化hexo博客工程：
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
4、编译博客系统：hexo g
5、运行hexo：hexo s
6、另外常用的hexo命令有：
部署：hexo dploy
清除缓存：hexo clean
升级hexo：npm update hexo -g

注：这样搭建好的博客，是在本地，而且是静态的

五、修改博客主题

1、访问hexo.io，点击themes，选择自己喜欢的主题，点击主题名（点图片是相当于预览）

2、安装（此命令，克隆主题到你的博客文件的themes文件夹下）
git clone https://github.com/Haojen/hexo-theme-Anisina.git themes/Anisina

修改_config.yml文件的themes属性为themes: Anisina

六、部署到Github

1、安装hexo的部署插件：npm install hexo-deployer-git –save

2、修改_config.yml文件, 在deploy下指定仓库路径和部署的协议，具体配置如下：
deploy:
type: git
repository: git@Github.com:xx-yyzz.github.io.git
branch: master

3、重新编译文件：hexo g

4、部署前（重点）：https://pages.github.com/ 

打开Git Bash Here，输入以下命令

1.克隆存储库,转到要存储项目的文件夹，并克隆新的存储库
Git clone https://github.com/username/username.github.io
(username 换成自己的github账户名)

2.进入文件夹，并添加index.html文件
cd username.github.io
echo “Hello World” > index.html

3.添加、提交、推送更改
git add –all
git commit -m “Initial commit”
git push -u origin master

5、命令提示窗口（管理员模式）部署,执行命令:hexo d
